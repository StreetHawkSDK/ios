handleDefinedUserInfo:withAction:treatAppAs:forNotificationType:

This method is called to filter code. A StreetHawk style remote notification must have positive int value code, if not this method will return without any effect. After filter code, it calls - (BOOL)handleDefinedCode:(int)code userInfo:(NSDictionary *)userInfo withAction:(SHNotificationAction)action treatAppAs:(SHAppFGBG)appFGBG to perform action.

- (BOOL)handleDefinedUserInfo:(NSDictionary *)userInfo withAction:(SHNotificationActionResult)action treatAppAs:(SHAppFGBG)appFGBG forNotificationType:(SHNotificationType)notificationType

Parameters

userInfo

Server remote notification payload, or local notification payload by response.

action

If response on notification directly here is the answer, at this time App state is inactive.

appFGBG

Treat this remote notification handled as FG or BG.

notificationType

An enum type indicate it’s from remote notification or local notification.

Return Value

YES if this notification is handleable by StreetHawk SDK; NO if it’s not recognizable or not as expected as StreetHawk SDK.
