SHRequest Class Reference

Inherits from NSOperation
Declared in StreetHawkRequest.h

Overview

All http requests used to communicate with server uses this class. It’s a wrapper of NSURLConnection and easier to use by block callback.

requestWithPath:withVersion:withParams:withMethod:withHeaders:withBodyOrStream:

Helper method to create a request object.

+ (SHRequest *)requestWithPath:(NSString *)path withVersion:(SHHostVersion)hostVersion withParams:(NSArray *)params withMethod:(NSString *)method withHeaders:(NSDictionary *)headers withBodyOrStream:(id)body_or_stream

Parameters

path

The internal path after root url, for example: “products/product_id/description/”.

hostVersion

The version of current host.

params

A key/value pair, which will format to paramters in URL, for example (@“family”, @“tops”) will become “?family=tops”.

method

GET or POST. If pass nil it’s GET by default.

headers

The header fields sent in request. Pass nil if no header need to set.

body_or_stream

Set to post body. It’s usually array, dictionary or data.

Return Value

An auto-released request.

cancel

Cancel the connection and dealloc it. Only asynchronous request can be cancelled. requestHandler is invoked with error=requestCancelledError. Note: Once a request is cancelled or finished, it cannot be added into queue again, otherwise exception occur.

startAsynchronously

Add request into operation queue. It will stay in the queue and start when ready to execute it, or caller to cancel it. Once it’s finished or cancelled it’s auto released. The result is callback by requestHandler.

- (void)startAsynchronously
